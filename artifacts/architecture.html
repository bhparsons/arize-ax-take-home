How one query flows through the agent

A single run.py question, tracing the round-trips: RAG retrieval → web search → final answer. Numbered badges ①–⑨ follow execution order. Everything inside the big panel runs in one Python process on your machine; the model, web, and trace store are three separate remote services reached over HTTPS. The telemetry channel (green, dashed) is fire-and-forget — the agent never waits on Arize.
